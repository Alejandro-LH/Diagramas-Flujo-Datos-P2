# Portafolio de Ejercicios con DFD y Código en Dart

Este repositorio reúne una colección de ejercicios resueltos en clase mediante diagramas de flujo de datos (DFD), análisis del problema, pruebas de escritorio y su implementación en Dart.

El contenido muestra el proceso completo de resolución: desde la lógica del algoritmo hasta su traducción a código, lo que lo convierte en un material útil tanto para estudiar diseño algorítmico como para practicar programación estructurada.

## Descripción

El material incluido en este repositorio presenta ejercicios académicos centrados en:

- análisis de problemas
- representación con DFD
- pruebas de escritorio
- identificación de entradas y salidas
- implementación en Dart
- uso de ciclos `for`, `while` y `do-while`
- trabajo con arreglos y estructuras básicas

## Archivo principal

- [Portafolio-2.md](/Users/alejandrolh/Downloads/Portafolio-2.md): documento principal con ejercicios resueltos, diagramas, análisis y código en Dart.

## Contenido del repositorio

Entre los temas desarrollados se encuentran:

- sumas acumuladas
- números pares
- almacenamiento de datos en arreglos
- repetición controlada con distintos tipos de ciclos
- ejercicios comparativos entre `for`, `while` y `do-while`
- lógica secuencial e iterativa aplicada en código

## Objetivo

El propósito de este repositorio es documentar la resolución paso a paso de distintos ejercicios académicos, mostrando la relación entre el diseño del algoritmo y su implementación práctica en Dart.

## Enfoque

Este repositorio tiene un enfoque académico y práctico. Por ello, combina documentación algorítmica con código fuente, permitiendo observar cómo una misma solución puede representarse visualmente, analizarse manualmente y programarse.

## Tecnologías y formato

- Markdown
- Dart
- Diagramas de flujo de datos (DFD)

## Cómo usar este repositorio

1. Revisa cada ejercicio en orden.
2. Lee el análisis para entender el planteamiento.
3. Observa el DFD para visualizar la lógica.
4. Consulta la prueba de escritorio para seguir el comportamiento del algoritmo.
5. Revisa el código en Dart para entender su implementación.

## Nota

Este material funciona como evidencia de aprendizaje y práctica académica, por lo que puede seguir creciendo con nuevos ejercicios, mejoras de redacción o más ejemplos en Dart.
